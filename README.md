# Library Management System

A console-based Library Management System built using Java to practice Object-Oriented Programming and Java Collections.  
The application allows users to manage books through a simple menu-driven interface.

## Features

- Add a new book to the library
- View all available books
- Borrow a book
- Return a borrowed book
- Search for a book by title
- Menu-driven console interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList (Java Collections Framework)
- Scanner for user input

## How It Works

1. The program displays a menu with different options.
2. Users can add books by entering book details such as ID, title, and author.
3. All books are stored in an ArrayList.
4. Users can borrow, return, or search for books.
5. The program continues running until the user selects the exit option.

## Concepts Practiced

- Classes and Objects
- Constructors
- Object state management
- ArrayList and collections
- Searching objects in a list
- Menu-driven program design

This project was built to strengthen Java fundamentals and gain hands-on experience implementing Object-Oriented Programming concepts in a practical application.
